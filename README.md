# CubicMC Launcher

CubicMC Launcher es un launcher moderno y elegante para Minecraft, desarrollado con Next.js y TypeScript. Este proyecto proporciona una interfaz de usuario intuitiva para gestionar y ejecutar diferentes versiones de Minecraft.

## Características

- 🎮 Interfaz de usuario moderna y responsive
- 🔄 Gestión de múltiples versiones de Minecraft
- 📦 Descarga automática de versiones y recursos
- 🎨 Temas personalizables
- 🌐 Multiplataforma (Windows, macOS, Linux)

## Requisitos Previos

- Node.js 18.0.0 o superior
- npm o yarn
- Git

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/cubicmc-launcher.git
cd cubicmc-launcher
```

2. Instala las dependencias:
```bash
npm install
# o
yarn install
```

3. Inicia el servidor de desarrollo:
```bash
npm run dev
# o
yarn dev
```

4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Tecnologías Utilizadas

- Next.js
- TypeScript
- Tailwind CSS
- Electron (para la versión de escritorio)
- React Icons
- Zustand (gestión de estado)
